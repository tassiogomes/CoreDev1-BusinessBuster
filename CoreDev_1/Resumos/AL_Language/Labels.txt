		• Utilizado em legendas, descrições, mensagens, etc.

	• Parâmetros opcionais:
		• Comment: Comentários gerais sobre o label, especialmente sobre os espaços reservados.
		• Locked: Booleano. Se true, o label não deve ser traduzido. Valor padrão é false.
		• MaxLength: Inteiro. Determina o comprimento máximo do label. Sem limite padrão se não especificado.

	• Sintaxe exemplo:
		• var a: Label 'Texto do Label', Comment='Comentário sobre o label', MaxLength=999, Locked=true;

	• Usos:
		• Propriedades de página e relatório:
			• Caption = 'Texto da legenda', Comment = 'Comentário sobre a legenda', Locked = false, MaxLength = 999;
		• Exemplo de uso em propriedades:
			• Caption property
			• ToolTip property
			• OptionCaption property

	• Exemplos:
		• Declaração de label:
			• var
				• myLabel: Label 'Este é um texto de label', Comment='Explicação sobre o texto', MaxLength=50, Locked=true;
		• Uso em uma página:
			• label(MyPageLabel)
				• {
					• ApplicationArea = Basic, Suite;
					• Caption = 'Texto exibido na página';
				• }
